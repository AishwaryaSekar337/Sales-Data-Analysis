#  E -commerce Sales Data Analysis using Python

## Project Overview

This project focuses on analyzing sales data using Python. The dataset was cleaned, processed, and analyzed to identify sales trends, revenue performance, product category performance, and regional sales distribution.

## Objectives
* Clean and preprocess the sales dataset
* Handle missing values and duplicate records
* Calculate total revenue
* Analyze monthly sales trends
* Analyze product category performance
* Analyze regional sales performance
* Visualize insights using charts and graphs

## Tools and Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Data Cleaning Steps
* Removed duplicate records
* Handled missing values
* Converted data into proper formats
* Created Total Revenue column

## Analysis Performed
### Revenue Analysis
* Calculated total revenue generated from sales

### Monthly Sales Analysis
* Identified sales trends across different months

### Product Category Analysis
* Compared revenue generated by different product categories

### Regional Analysis
* Evaluated sales performance across regions

## Visualizations
* Monthly Revenue Trends
* Product Category Revenue Chart
* Regional Sales Distribution

## Key Insights

* Identified top-performing product categories
* Determined highest revenue-generating regions
* Observed monthly sales patterns
* Generated business insights from the dataset

## Conclusion
This project demonstrates the use of Python for data cleaning, exploratory data analysis (EDA), and data visualization to support business decision-making.
